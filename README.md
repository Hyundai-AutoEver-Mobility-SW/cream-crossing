# Cream Crossing
React 상태관리 복습과 피그마 디자인 실습을 위한 프로젝트
- Redux를 이용한 상태관리
- React Router를 통한 라우팅
- 카카오맵 API를 통해 사용자 주소와 위치 업데이트
- CSS : tailwind, styled-component
